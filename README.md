# Mind Orbit

Mind Orbit is a tiny local beta for turning notes into orbiting keyword nodes.

## What it does

- Type a memo.
- The app extracts 5 simple keywords.
- Keywords appear as orbiting nodes.
- Click a node to read the original memo.
- Memos are saved in the browser with localStorage.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000 in your browser.

## How to use

1. Type a memo in the left text box.
2. Click "키워드 노드 만들기".
3. Click a round keyword node in the center orbit map.
4. Read the saved memo in the right panel.
5. Refresh the page to confirm that the memo stays saved.
