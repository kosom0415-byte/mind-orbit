# Human Approval Policy

Human Vision Owner is responsible for direction, approval, rejection, and risk acceptance.

## Human Approval Required
- Production deploy
- Rollback
- env/API key changes
- Billing/account actions
- HIGH/CRITICAL task approval
- Product direction
- Design direction
- App runtime architecture changes
- Auth/security/payment/database schema changes

## AI Can Handle
- Log summary
- Task generation
- Blocked reason analysis
- Retry proposal
- Codebase analysis
- Report generation
- Safe task handoff

## Response Choices
- approve
- reject
- modify-scope
- ask-gpt

No approval is implied by silence.
